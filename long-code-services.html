Long Code SMS Service

Long Code is one of the effective ways of conveying information to the prospective customers. Long Code is a 10 digit virtual number which is used for text based information. The keyword request received from various customers are recorded to understand their choices and preferences. Post receiving the request, facility to acknowledge the customer is probable done with an automated reply.

It is mostly used for mobile verification systems, lead generation, opt-in database systems, mobile ticketing and SMS subscription activation. The information is provided based on the keyword request. Single number can be used for multiple purposes with no human intervention that will enable businesses to reach their customers instantly.

Long Code SMS service is used for receiving feedback from the customers and mostly used in banking, hospital, educational institutions etc.

Long codes are provided in two different ways for enterprises.

Dedicated Long Code SMS

Generally a Dedicated Long Code gets assigned to a particular organization completely and it cannot be used by any other user. Unlimited keywords can be availed based on the business criteria.

Shared Long Code SMS Service

In this scenario, one specific Long Code can be shared with multiple organizations having the limit on keywords. Based on the assigned keyword, enterprises can segregate their customer request.

Long Code Reply

Post request of the customer through Long Code, a customized automated reply is sent as an acknowledgement.
