Overview

Short URL displays data via an Application Programming Interface (API). Developer’s do the backend coding to generate the unlimited Short URL’s as per the requirement. This API is customizable and configurable as per the client’s business need.

Short URL API Work Flow

API Reference

Request

http://ion.bz/shortcodeAPI.php?api_key=2c3f9ea2-49de-4d52-a662-1feecc40506b&user_url=www.smsstriker.com

Response

http://ion.bz/js24rkw

Input Parameters Description

Parameter Description Required
api_key API Key Assigned by Striker Yes
user_url Custom URL which to be converted as Short URL Yes
