API Reference

Request

http://yourweburl.com/request.php?Phone=00000000&ServiceNo=04000000000&Message=hello 00000000&smsTime=YYYY-MM-DD HH:MM

Response

Array("Phone" =>"918886600000000",
"ServiceNo" =>"83670000XX ",
"smsTime" => "08/05/17 06:01:45 PM ",
"Message" => "EXAMS918886638814" )

Input Parameters Description

Parameter Description Required
Your URL The URL from which you need information, that URL should mention as input box of Long Code Tab on SMS Striker panel Yes
Phone Second parameter is phone number and this parameter can be customized the way you want to display parameter as per choice. Yes
Service No Third parameter should be Long Code number and this parameter can be customized the way you want display parameter as per choice. No
sms Time Fourth parameter should be timestamp for received message this parameter can be customized the way you want display parameter as per choice. No
Message Fifth paratmer should be message for received text and this parameter can be customized the way you want display parameter as per choice. Yes
