Long Code

Long codes are one of the effective ways of conveying the information to your clients and customers. As per this concept a 10-digit virtual number is used to send information which is text-based. The information received is recorded to understand the choices and preferences of the customer. It is mostly used for mobile verification systems, lead generation, opt-in database systems, mobile ticketing and SMS subscription activation.

Long codes are provided in two different ways for enterprises.

  • Dedicated Long Code
  • Shared Long Code
